# desktop-client

## Prerequisites
* Node
* Electron
* Bower

#### Install Electron

    npm install -g electron
    
Check whether your electron works correctly

    electron --version

#### Install bower

    npm install -g bower

## Run the project

Run following commands inside the root folder of the project where package.json located at. 

    npm install
    bower install
    
Run the app

    electron ./main.js
